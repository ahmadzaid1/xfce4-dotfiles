## Usage

To use these configuration files:

1. Clone or download this repository to your local machine.

2. Navigate to the `.config/xfce4` directory in your home directory.

3. Replace the existing configuration files with the ones from this repository.

4. Restart the XFCE desktop environment, logging out and logging back in would be enough.
